# 雙月筆記應用程式設計規格

## 專案概述

設計一個單一 HTML 檔案應用程式，提供月曆檢視與筆記功能，所有資料儲存在本地。

## 功能需求

### 1. 介面設計

- **整體風格**
  - 簡潔、輕量的設計
  - 背景顏色：深灰色
  - 響應式設計，適應不同螢幕尺寸

- **版面配置**
  - 左側：月曆顯示區域
  - 右側：筆記輸入與顯示區域
  - 左右分欄布局

### 2. 月曆功能

- **顯示內容**
  - 垂直排列顯示兩個月曆
    - 上方：當月月曆
    - 下方：下個月月曆
  - 每個月曆顯示完整的月份日期網格
  - 標示當天日期（高亮顯示）
  - 週六、週日使用特殊底色標示

- **月份導航**
  - 提供月份切換控制按鈕
  - 可切換到上一個月或下一個月
  - 提供「回到今天」按鈕快速回到當前月份
  - 切換月份時，兩個月曆同步更新顯示

- **國定假日功能**
  - 在對應日期下方標註假日名稱（紅色小字）
  - 支援匯入自訂假日資料（JSON 格式）
  - 透過三點選單存取匯入/匯出功能
  - 匯入：上傳 JSON 檔案，覆蓋模式，需確認
  - 匯出：下載為 JSON 檔案，檔名格式 `holidays-YYYY-MM-DD.json`
  - 資料自動儲存在 LocalStorage，下次開啟自動載入

### 3. 筆記功能

- **輸入與編輯**
  - 右側提供多行文字輸入區域
  - 輸入後 500ms 自動儲存

- **匯入/匯出**
  - 透過三點選單存取匯入/匯出功能
  - 匯入：上傳文字檔案（.txt），覆蓋模式，需確認
  - 匯出：下載為文字檔案，檔名格式 `notes-YYYY-MM-DD.txt`
  - 資料自動儲存在 LocalStorage，下次開啟自動載入

## UI/UX 考量

- **色彩配置**
  - 深灰色背景（#2b2b2b）
  - 淺色文字確保可讀性
  - 週六、週日使用紅色系標示（紅色文字 + 半透明紅色背景）
  - 假日名稱使用紅色小字顯示在日期下方

- **互動設計**
  - 筆記輸入框即時儲存（輸入後 500ms 自動儲存）
  - 匯入匯出功能透過三點（⋯）圖示按鈕和下拉選單存取
  - 三點選單位於「月曆」和「筆記」標題旁
  - 匯入失敗時使用 alert 顯示錯誤訊息
